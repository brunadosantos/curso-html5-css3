Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b> (não semântica)

Nessa frase temos um termo em destaque usando a tag <strong> (semântica)

Itálico / Ênfase

Nessa frase temos um termo em itálico usando a tag <i> (não semântica)

Nessa frase temos um termo em ênfase o usando a tag <em>(semântica)

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark>

Nesse paragrafo temos um texto marcado usando a tag <mark>

Testo Grande e Pequeno

Estamos criando um texto grande usando a tag <big> (obsoleto) e um texto pequeno usando a tag <small> nesse parágrafo (semântica)

Texto Deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado usando a tag <del> (semântica)

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois usando a tag <ins> ( semântica)

Existe também o texto sublinhado com a tag <u> (não semântica)

Texto Sobrescrito

Usado para inserir textos do tipo X20+3 usando a tag <sub> (semântica)

Texto Subescrito

Usado para inserir textos do tipo H2O usando a tag <sup> (semântica)
